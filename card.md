# 카드 25장 초안

이 문서는 현재 시스템 기준으로 사용할 수 있는 25장 카드 풀 초안이다.

기준:

- 종족 5개: 인간, 엘프, 오크, 마족, 언데드
- 직업 5개: 전사, 탱커, 궁수, 마법사, 암살자
- 각 종족과 직업 조합을 1장씩 배치해 총 25장 구성
- 스탯은 1차 초안이며 이후 난이도 조절 단계에서 수정 가능

카드 구조:

```ts
type Card = {
  id: string;
  name: string;
  race: "human" | "elf" | "orc" | "demon" | "undead";
  job: "warrior" | "tank" | "archer" | "mage" | "assassin";
  attack: number;
  hp: number;
  attackType: "melee" | "range" | "magic";
}
```

## 정리

- 총 카드 수: 25장
- 종족별 카드 수: 5장
- 직업별 카드 수: 5장
- 모든 종족-직업 조합이 1장씩 존재

## 다음 고민 포인트

- 시작 덱에 어떤 카드들을 몇 장씩 넣을지
- 카드 등급별 스탯 편차를 얼마나 줄지
- 특정 공격 타입이 과하게 강하지 않은지
- 난이도 조절을 위해 적 체력/공격력과 카드 평균 스탯을 어떻게 맞출지
